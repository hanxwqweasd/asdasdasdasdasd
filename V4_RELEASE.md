# V4 Release Verification

## Реализованные системы

Все 17 продуктовых пунктов включены в клиент, API, PostgreSQL-схему и админский LiveOps-контур. Платные главы и кооперативные дела проверяют entitlement на сервере.

## Автоматическая проверка

```bash
npm ci
npm run check
npm test
npm run build
node --check public/app.js
node --check public/admin/admin.js
```

## Нагрузочный стенд

- 2 000 HTTP-запросов от 1 000 пользователей: 2 000 успешно, 0 ошибок.
- 1 000 одновременных Socket.IO-клиентов: 1 000 успешно, 0 ошибок.
- 100 параллельных кооперативных матчей / 200 игроков: 100 успешно, 0 ошибок.
- наблюдатель: вмешательство, блокировка второго вмешательства и reconnect успешно.

## Интеграционная проверка

- миграции применены на PostgreSQL 17 повторно без ошибок;
- Redis 8 использован для realtime и восстановления;
- сюжетная глава из 14 сцен пройдена до конца через HTTP API;
- финальная награда выдана один раз;
- платное дело создал владелец, два гостевых места разрешены, комнаты соответствуют сценарию;
- backup восстановлен в пустую базу и прошёл SHA-256;
- Stars fulfillment/refund проверены на идемпотентность серверного контура;
- все аудиофайлы читаются `ffprobe`;
- SVG-ссылки и CSS-структура проверены;
- `npm audit --omit=dev` — 0 уязвимостей.

Нативные окна Telegram и реальное списание Stars требуют финального smoke-test в настоящем боте после Railway deploy.
