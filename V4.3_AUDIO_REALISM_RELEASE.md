# V4.3.1 — Natural Audio Release

- removed all button, tab, slider and modal sounds;
- rebuilt 79 world and ambience assets at 48 kHz stereo;
- softened peaks and high-frequency transients;
- replaced game-like stingers with physical paper, relay, door, air and building sounds;
- added distance filtering and natural room reflections;
- reduced random house-event frequency and volume;
- changed default mix to 64% ambience and 52% world actions;
- removed the interface-volume control because the interface is silent.
